- **AgentConfigLoader**: `config/agents/*.yml` 스캔 → AgentDefinition 파싱

- **PipelineService**: 통합 서비스 (agentId 기반 라우팅, 실행별 OrchestratorClient)
- **SyncDataSourceService**: RSV + Loader 버전 머지 (ThreadLocal 기반)
- **DynamicEntityManagerService**: Loader에서 복사 (JPA EntityManager 동적 생성)
- Entity 12개: sync-entities-dmz에서 흡수 (패키지 변경)
- Step/Fetcher 복사: LinkTableObsvDataFetcher, LinkTableUpdateStep, DaejeonLoadStep, TargetRepositoryService
- Controller: PipelineController (통합), HealthController (모든 Agent 표시)

### Phase 4: ExecutionData 컨트롤러

### Phase 5: RCV 10개 확장

- 총 12개 Agent YAML (RCV 10 + Loader 1 + SND 1)
