## 아키텍처 변경 요약
```
AS-IS:
  Pipeline Agent (8082) = 파이프라인 실행 + DB 프록시 (혼합)
  Orchestrator → Pipeline Agent → DB 조회

TO-BE:
  Pipeline Agent (8082/8092) = 파이프라인 실행만
  Proxy Agent (8083/8093) = DB 프록시만
  Orchestrator → Proxy Agent → DB 조회 (zone_config.proxy_agent_url 기반)
  Orchestrator → Pipeline Agent → 파이프라인 트리거 (agent.endpoint_url 기반)
```
