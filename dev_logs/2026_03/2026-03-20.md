**문제**: DMZ Loader가 IF→Target 복사 시 source_refs를 그대로 복사해서, target의 source_refs가 외부 DB PK를 가리킴.
backward trace 시 IF 테이블의 id로 매칭해야 하는데, 외부 DB PK로 조회해서 0건.

**수정**:
- `LoaderStepHelper.processJewon/processObsvdata`에 `StepContext` 파라미터 추가

- `DmzBojoLoadStep`에서 context 전달
