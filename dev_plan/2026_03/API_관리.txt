Agent관리와 비슷한 기능을 가지지만 최초 datasoruce를 외부 API 호출로 하는 데이터 소스 관리 기능임
DMZ와 내부망에서 각각 외부 호출을 해서 각자의 DB에 데이터를 쌓음
DMZ에 쌓인 데이터는 최종적으로는 내부망으로 옮겨놓는것까지가 목표

외부 API에서 쓰는 파라미터를 자체 db에 저장해서 관리하고 api 호출에 schedule을 달아서 주기적으로 데이터를 쌓도록함
//여기서 메타 데이터 가공 기능이 필요함, 이는 미리 정의된 가공방식만을 제공하며 예를 들면 날짜면 20220102, 2022-02-02 등의 제공방식을 

1차적으로 api 테스트를 외부에 쏴서 return 형태에 대한 정보를 얻음(아마도 json아니면 xml일것)
그럼 각각 항목에 대한 값을 적재할 db 컬럼에 mapping을 해줌
이 db들은 이전에 만들어둔 db관리에서 등록된 tb와 col 대상으로 이루어질것임

Agent와 유사하게 등록을 하고 스케줄러로 돌려놓는 방식이 기본 활용방법임

Agent관리처럼 API 관리라는 큰 탭을 만들고 외부 API 정보 수집(지금 만드는거) / 내부 API 제공(추후개발) 로 메뉴 구분을 함
