# 6. 프로시저 관리

> **요구사항**: DB 내 프로시저를 게시판 형태로 관리하고,
> 프로시저 문을 암호화하여 안전하게 제공한다.

## 상태: 미개발 (요구사항 미확정)

---

## 요구사항 확정 [확정필요]
- [ ] 대상 범위: 어떤 DB의 프로시저를 관리할 것인지
- [ ] 암호화 방식: 조회 시 복호화 제공 여부, 권한 체계
- [ ] 버전 관리: 프로시저 변경 이력 추적 여부
- [ ] 실행 기능: 등록된 프로시저를 UI에서 직접 실행 가능 여부

## 프로시저 게시판 [Backend]
- [ ] 프로시저 등록/수정/삭제
- [ ] 프로시저 문 암호화 저장
- [ ] 조회 시 복호화 제공

## 프로시저 관리 화면 [Frontend]
- [ ] 게시판 형태 목록/상세
- [ ] 프로시저 문 등록/편집
