# 5. API 제공

> **요구사항**: GIMS DB에 적재된 데이터를 외부 시스템에 API로 제공한다.
> UI에서 오퍼레이션을 등록하면 동적 SELECT 쿼리가 생성되어 API로 노출된다.
> 테이블 데이터는 이미 가공된 상태로 존재하며, 조인 등 복잡한 쿼리는 범위 밖.

## 상태: 미개발

---

## 동적 쿼리 범위 [참고]
- SELECT 컬럼: 제공할 컬럼 선택
- WHERE 조건: 필터 조건 설정
- ORDER BY: 정렬 기준
- LIMIT: 응답 건수 제한
- 범위 밖: JOIN, INSERT/UPDATE/DELETE, 서브쿼리, 집계함수

## 오퍼레이션 관리 [Backend]
- [ ] 오퍼레이션 등록 (Datasource + 테이블 선택)
- [ ] 제공 컬럼 선택
- [ ] WHERE 조건 설정
- [ ] 정렬/LIMIT 설정
- [ ] 응답 포맷 선택 (JSON, XML 등)

## 테스트 호출 [Backend]
- [ ] 등록한 오퍼레이션 미리보기 (실제 쿼리 실행 → 결과 샘플 표시)
- [ ] 결과 확인 후 "게시" — 게시 전에는 외부 노출 안 됨
- [ ] 게시/비게시 상태 관리

## API 엔드포인트 생성 [Backend]

- [ ] 인증/인가 (API Key, IP 제한 등)
